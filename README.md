# Avery Stone Portfolio

A polished personal portfolio built for a product engineer. The site presents selected projects, resume content, writing, and a Netlify-powered contact form.

## Key Technologies

- TanStack Start and TanStack Router for the React application
- Content Collections for markdown-backed blog, project, resume, and education content
- Tailwind CSS v4 for styling
- Netlify Forms for contact submissions
- Netlify's TanStack Start Vite plugin for deployment support

## Run Locally

Install dependencies, then start the development server:

```bash
npm install
npm run dev
```

The app runs on the port configured by the development command in `package.json`.
